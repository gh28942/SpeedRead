# SpeedRead
A speedreading app written in Java. Features loading from txt &amp; URLs, and bookmarking text to read.

<ul>
<li>Written in Java</li>
<li>Read text from plain text files</li>
<li>Read text from URLs (get Website text)</li>
<li>Read at different speed levels</li>
<li>Black/White design modes</li>
<li>Useful for long texts: Set a bookmark to read remaining text later</li>
<li>See how long (in minutes) it will take to speedread a text</li>
<li>Stop the flow of text by tapping on the screen</li>
<li>English and German language (specified in the strings.xml files)</li>
<li>Implements the <a href="https://github.com/gh28942/Android-default-offline-CrossPromo-Ad">offline ad</a>.</li>
</ul>
